### Run Locally

To run locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/ajkun55/Tech-Homepage.git
```

Windows:

```bash
git clone https://github.com/ajkun55/Tech-Homepage.git
```
